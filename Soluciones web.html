Soluciones Web

Son parametros que utilizamos para medir la factibilidad de uso de una pagina web

Usabilidad

Se refiere a la facilidad de uso que tiene una pagina o aplicacion web
La usabilidad tiene 3 factores:

  • Eficiencia:El usuario satisface la necesidad de su busqueda
  • Eficacia:El usuario logra rapidamente alcanzar su cometido
  • Satisfaccion:El usuario se sintio agradable al navegar en el sitio web

¿Porque es importante la usabilidad en una pagina web?

La falta de usabilidad web es la mayor causa de abandono de los usuarios

Interactividad

Un sitio web interactivo es aquel en que los usuarios pueden interactuar con diferentes elementos y tener una experiencia de usuario que no se limita hacer click y desplazarse.Estos son hechos por diseñadores y desarrolladores cuyo trabajo principal es ofrecer un sentido de Personalizacion

¿Cuales son los beneficios de un sitio web interactivo?

  1. Aumenta proporcionalmente la participacion del usuario y ayuda a retener su atencion
  2. Es propabla que los usuares pasen mas tiempo en el sitio web
  3. Los sitios web interactivos brindan una experiencia de usuario personalizadas y ayudan a aumentar el numero de clientes satisfechos
  4. Crea un impacto positivo en los usuarios y generan un buen recuerdo
  5. Existe mayor probabilidad de que los usuarios recomienden el sitio web a otros

Funciones Interactivas

  1. Google Maps:Proporciona una buena forma de mejorar la interacion del usuario y aumenta la confianza
  2. Crear Votaciones:Mostrar los resultados de las encuestas ayuda a aumentar la confianza y el compromiso
  3. Funciones de la Barra de Busquedad:dentro del sitio ayudan al usuario a navegar de manera mas efectiva.Funciones de filtrado, De ordenacion, entre otras
  4. Calificaciones:Son una forma visual rapida de obtener comentarios para mejorar los productos y servicios

Elementos Esenciales

A la hora de elaborar una pagina web que sea usable debemos tener en cuenta 4 grandes elementos cuya estructura y aspecto van a determinar dicha usabilidad

Identidad

Es la personalidad que refleja el sitio.Aqui es importante que la home describa

Accesibilidad

Es la posibilidad de que todos los individuos puedan hacer uso de la web con independencia de las limitaciones del individuo o del dispositivo

Navegacion

Entendemos como tal a la facilidad de desplazarse por las distintas paginas que componen la web.El usuario debe saber en todo momento en que parte de la web se encuentra y como ir a otras secciones

Contenido

El contenido que se ofrece sera conciso, concreto y sencillo. Debemos utilizar un lenguaje propio del publico objetivo "target", asi como ofrecer una información de valor para el mismo
