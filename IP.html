¿Que es una Direccion IP y como funciona?

Los protocolos de internet (Internet Protocol) son las anclas de internet, conectan a la red mundial y apuntala toda su vida digital.De hecho, es imposible conectarse a internet sin una direccion IP

Analisis de las direcciones IP

Suelen consistir en 4 numeros del 0 al 255, separados por puntos.Dentro de una direccion IP, Puede ver el ID de Red, asignado a su red y el ID del host, el identificador unico asignado a cada dispositivo conectado a esa red específica.


Este es un ejemplo de direccion IP:192.168. 1.3

ID de Red

Es la parte de una direccion IP con la que se identifica la red que se esta utilizando para conectarse a internet, es decir los 3 primeros numeros

ID del host(anfitrion)

Una red puede ser tan pequeña como dos ordenadores entre si o tan grande como internet en si mismo, la ID del anfitrion indica el dispositivo concreto que esta usando para conectarse a un Red

Ejemplo

192.168. 1.3

Los numeros de color azul representan al ID de la red y los numeros de color rojo representan el ID del host
