Sitios Web

Son sitios que contienen paginas web organizadas jerarquicamente.Son espacios virtuales que contienen toda la informacion que una persona o empresa quiere difundir atravez de internet

Diferencia entre un sitio web y una pagina web

Una pagina web es un documento electronico donde podemos encontrar:

  • Texto
  • Videos
  • Imagenes
  • Audios
  • Animaciones

Por su parte un sitio web es una coleccion de paginas web relacionadas entre si
